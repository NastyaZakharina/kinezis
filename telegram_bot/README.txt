========================================
  TELEGRAM-БОТ КІНЕЗІС — ІНСТРУКЦІЯ
========================================

КРОК 1 — Реєстрація бота (2 хвилини)
--------------------------------------
1. Відкрийте Telegram
2. Знайдіть @BotFather і напишіть /newbot
3. Введіть назву бота: Кінезіс
4. Введіть username: KinezisShopBot
   (якщо зайнятий — спробуйте KinezisUA_Bot)
5. BotFather надасть TOKEN — скопіюйте його

КРОК 2 — Отримайте свій Telegram ID
--------------------------------------
1. Знайдіть @userinfobot в Telegram
2. Натисніть /start
3. Бот покаже ваш ID (число) — збережіть
4. Те саме зробіть Андрій

КРОК 3 — Налаштуйте bot.py
--------------------------------------
Відкрийте файл bot.py і замініть:

  BOT_TOKEN = "ВСТАВТЕ_ВАШ_ТОКЕН_ВІД_BOTFATHER"
  → вставте токен з кроку 1

  MANAGER_CHAT_IDS = [123456789, 987654321]
  → замініть числа на ID з кроку 2

КРОК 4 — Встановіть Python
--------------------------------------
1. Завантажте: https://www.python.org/downloads/
2. Встановіть (обов'язково поставте галочку "Add to PATH")
3. Відкрийте командний рядок (Win+R → cmd)
4. Введіть: pip install pytelegrambotapi

КРОК 5 — Запуск бота
--------------------------------------
1. Відкрийте командний рядок в папці telegram_bot
2. Введіть: python bot.py
3. Бот запрацює — тримайте комп'ютер увімкненим!

КРОК 6 — Оновіть посилання на сайті
--------------------------------------
Коли отримаєте username бота від BotFather,
замініть "KinezisBot" на реальний username
у всіх HTML-файлах сайту.

Наприклад: https://t.me/KinezisShopBot

АВТОЗАПУСК (необов'язково)
--------------------------------------
Щоб бот запускався автоматично при старті Windows:
1. Створіть файл start_bot.bat:
   cd C:\Users\user\Kinezis\telegram_bot
   python bot.py
2. Покладіть ярлик у папку Автозавантаження:
   Win+R → shell:startup → вставте ярлик
